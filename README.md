## Setup Instructions

1. Clone the repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Set up the frontend:
   ```bash
   cd frontend
   npm install
   ```
4. Run the application:
   ```bash
   # Terminal 1 - Backend
   python app.py
   
   # Terminal 2 - Frontend
   cd frontend
   npm start
   ```
